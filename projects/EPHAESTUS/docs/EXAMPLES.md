# Hephaestus – Example Prompts

- **Product showcase**
  - “Create a product showcase for a luxury watch: add a plane floor, three-point lighting, isometric camera, and set viewport screenshot.”
- **Concrete base**
  - “Add a concrete material preset to the selected object and bump roughness to 0.9.”
- **City block**
  - “Create 5 cubes as buildings, randomize heights, and array street lamps along X.”
- **Studio shot**
  - “Create a camera named RenderCam at (6, -6, 4), apply the product preset, set active camera, apply studio lighting preset.”
- **Boolean cut**
  - “Add a cylinder, place it through the cube, and run a boolean difference on the cube with the cylinder.”
